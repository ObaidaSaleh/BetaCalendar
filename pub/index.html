BetaCalendar - Create simple yet appealing organization widgets

Welcome to BetaCalendar! This library is designed to provide developers with easy to use organization-based widgets. Take a look at the sidebar to the left to see various examples of BetaCalendar, or go to the documentation page to learn how BetaCalendar works!

The code behind the examples:

Simple Calendar With No TodoList

      
        const calendar1 = new BetaCalendar("#normalCalendar");

Minimizable Calendar

      
        const calendar2 = new BetaCalendar("#minimizableCalendar",

                                            "basic", 
                                            true);

        const calendar4 = new BetaCalendar("#importedRemindersCalendar",
